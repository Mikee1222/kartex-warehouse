# Kartex Αποθήκη

Warehouse app for Kartex (picking, inventory, schedule). Built with Next.js and Supabase.

## Development

```bash
pnpm install
pnpm dev
```

Runs at [http://localhost:3001](http://localhost:3001) (see `package.json`).

Copy `.env.local.example` to `.env.local` and set Supabase URL + anon key.

## Scripts

- `pnpm dev` — development server
- `pnpm build` — production build
- `pnpm start` — production server
